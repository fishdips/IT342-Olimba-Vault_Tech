# VaultTech

A Spring Boot application featuring user authentication. 

## Current Features
* User Registration
* User Login
* Secure Password Hashing (BCrypt)

## Tech Stack
* Java
* Spring Boot
* Spring Data JPA
* Supabase

## How to Run
1. Clone the repository.
2. Update the `application.properties` with your database credentials.
3. Run the application using your IDE or via Maven/Gradle.
